How to find the length of linked list both iterative way and recursive way

1. Iterative way

int getCount(Node* head){
    int count=0;
    Node*current=head;
    while(current!=NULL){
        count++;
        current=current->next;
    }
    return count;
}

2. Recursive Way
algorithm

if head is NULL return 0
else return 1+getCount(head->next)
